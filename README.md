# Metaballo

> _μεταβάλλω_ &mdash; to change, to turn about, to transform.

A portfolio website for **Metaballo**, a fictional high-end Mediterranean-Greek restaurant and cocktail bar. The concept revolves around transformation &mdash; raw flame and spirit refined into experience &mdash; with a focus on premium steak, seafood (crayfish, prawns), and a curated cocktail programme.

## Project structure

```
Metaballo/
├── index.html       # Landing page (hero, philosophy)
├── assets/          # Photography &mdash; hero.jpg and other images
├── .gitignore
└── README.md
```

## Pages planned

- **Heritage** &mdash; the story, the philosophy of transformation, the team
- **The Grill** &mdash; menu featuring premium steak, lamb, Mediterranean cuts
- **The Bar** &mdash; cocktail programme, spirits, wine list
- **Reservations** &mdash; booking flow

## Design system

- **Typeface (display):** Cormorant Garamond &mdash; italic for hero, regular for Greek titling
- **Typeface (UI):** Inter &mdash; tracked uppercase for nav and small labels
- **Palette:**
  - Background `#0d0807` (warm near-black)
  - Cream `#ead8c6`
  - Peach accent `#f1b89a`

## Running locally

Open `index.html` directly in a browser, or use VS Code's _Live Server_ extension for hot-reload on edits.

## Hero image

Place your hero photograph at `assets/hero.jpg`. The CSS references this path.
